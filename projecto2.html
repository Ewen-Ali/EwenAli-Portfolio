Projecto 2
Instalação e Configuração de Rede Doméstica

Este projeto consistiu na instalação e configuração de uma rede doméstica completa, com acesso à Internet para múltiplos dispositivos. O objetivo foi aprender sobre roteadores, switches, endereçamento IP e conectividade básica.

Linguagens de programação usadas:

  • HTML5
  • CSS3

Tecnologias e Equipamentos Utilizados

  • Roteador Wi-Fi
  • Switch de rede
  • Cabos Ethernet
  • Configuração de endereçamento IP (DHCP/Static)

Objetivo do Projeto

  • Compreender o funcionamento básico de redes locais (LAN);
  • Aprender a configurar dispositivos de rede para comunicação segura;
  • Garantir conectividade estável e acesso à Internet para todos os dispositivos;
  • Documentar a topologia e configuração da rede.

Resultado Final

O resultado foi uma rede doméstica funcional e organizada, permitindo que todos os dispositivos conectados acedam à Internet com estabilidade e segurança. O projeto também serviu como prática de configuração de dispositivos de rede e gestão de tráfego local.
